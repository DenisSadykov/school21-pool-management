# Автовыгрузка платформы в оформленную Google-таблицу

Платформа является единственным источником данных. Apps Script записывает
актуальный снимок бассейна прямо в существующие оформленные листы:

- `volunteers` — состав бассейна, трайб-мастера и проверяющие групповые;
- `shifts` — записи волонтёров в готовые слоты расписания;
- `penalty` — пенальти учеников в `A:G`;
- `tribe_event` — участия учеников в трайб-мероприятиях в `A:I`;
- `tribe_events` — календарь трайб-мероприятий.

Справочник имён в `H:I`, подписи трайбов в `C`, формулы, оформление,
объединения, проверки данных и сводные блоки листов сохраняются. Платформа
обновляет в `volunteers` только рабочие диапазоны `A:B`, `D` и `F`.

## Подключение таблицы

1. Открой таблицу и выбери **Расширения → Apps Script**.
2. Замени содержимое `Code.gs` файлом из этой папки.
3. В `SECRET` укажи значение, совпадающее с `SYNC_SECRET` backend.
4. Выбери **Развернуть → Новое развёртывание → Веб-приложение**.
5. Запускать от своего аккаунта, доступ предоставить всем, у кого есть URL.
6. Скопируй URL вида `https://script.google.com/macros/s/.../exec`.
7. В настройках активного бассейна укажи ссылку на таблицу и URL Apps Script,
   затем включи автовыгрузку.

Backend отправляет полный снимок конкретного бассейна каждые пять минут. Полный
снимок устраняет дубли и самостоятельно исправляет расхождения после временных
ошибок. Изменения, сделанные вручную в управляемых диапазонах, могут быть
заменены следующим снимком.
