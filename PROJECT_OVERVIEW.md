# 📖 Обзор проекта

## 🎯 Цель
Создать современную, красивую систему управления бассейном для School 21, которая будет удобна для волонтёров, Team Leads и администраторов.

## ✨ Ключевые особенности

### 🎨 Дизайн
- **Тема School 21**: чёрный фон с зелёными акцентами (#00ff00)
- **Минималистичный интерфейс**: чистый, понятный, без лишних деталей
- **Адаптивный дизайн**: работает на телефонах, планшетах, ПК
- **Dark mode**: по умолчанию (как школа 21)

### 🔄 Функциональность
- **Управление сменами**: создание, редактирование, удаление
- **Волонтёры**: просмотр профилей, статистики, активности
- **Система вознаграждений**: автоматический учёт коинов и штрафов
- **Google Sheets синхронизация**: двусторонний обмен данными
- **Разные роли**: волонтёр, Team Lead, администратор
- **Статистика и отчёты**: аналитика по сменам

### 🚀 Техническая архитектура
- **Frontend**: React 18 с Modern CSS (переменные, grid, flexbox)
- **Backend**: Flask (Python) с REST API
- **БД**: SQLite (простая, встроенная)
- **Auth**: простая аутентификация по имени (можно расширить)
- **API**: полностью задокументировано

### 🔐 Безопасность
- CORS настроен правильно
- Environment переменные для конфиденциальных данных
- Нет хардкода API ключей
- Git ignore для .env и google_key.json

### 🆓 Бесплатность
- Все компоненты open source
- Может работать на бесплатных платформах (Vercel, Railway, Render)
- Нет платных зависимостей
- Легко развернуть на собственном сервере

## 🔄 Workflow использования

### Для волонтёра:
1. Открывает приложение
2. Входит по имени
3. Видит доступные смены
4. Выбирает смену для дежурства
5. Видит свою статистику

### Для Team Lead:
1. Входит с ролью "team_lead"
2. Создаёт/редактирует смены
3. Управляет волонтёрами
4. Видит статистику
5. Синхронизирует с Google Sheets

### Для администратора:
1. Входит с ролью "admin"
2. Имеет доступ ко всему
3. Может сбросить БД (осторожно!)
4. Экспортирует данные
5. Управляет синхронизацией

## 🚀 Развёртывание

### Локально (для разработки):
```bash
# Terminal 1 - Backend
cd backend && python app.py

# Terminal 2 - Frontend
cd frontend && npm start
```

### Docker (всё в одной команде):
```bash
docker-compose up
```

### На сервере:
- Frontend → Vercel (бесплатно)
- Backend → Railway/Render (бесплатно)
- Database → PostgreSQL на сервере

## 📊 Примеры API

### Создать смену
```http
POST /api/shifts
{
  "date": "2026-06-10",
  "time_start": "10:00",
  "time_end": "14:00"
}
```

### Получить волонтёров
```http
GET /api/volunteers
```

### Синхронизировать
```http
POST /api/sync
```
