# 👨‍💼 Гайд для Team Lead

Эта инструкция для тех, кто управляет системой бассейна — редактирует смены, работает с волонтёрами и отслеживает данные.

## 🎯 Основные задачи

### 1️⃣ Управление сменами

**Где?** Вкладка "Смены" → "Управление"

**Что можно делать:**
- ➕ Добавить новую смену
- ✏️ Отредактировать время, место, волонтёра
- 🗑️ Удалить смену
- 📋 Просмотреть историю изменений

**Пример:**
```
Дата: 10 июня 2026
Время: 10:00–14:00
Место: Бассейн (корпус A)
Волонтёры: didielsy, edithart
Статус: Подтверждено
```

### 2️⃣ Управление волонтёрами

**Где?** Вкладка "Волонтёры" → "Список"

**Что можно делать:**
- 👁️ Просмотреть профиль волонтёра
- 📊 Посмотреть статистику (сколько смен, коины, штрафы)
- ✏️ Редактировать информацию
- 🚫 Заблокировать/разблокировать доступ

### 3️⃣ Система коинов и штрафов

**Где?** Вкладка "Награды" → "Калькулятор"

**Как это работает:**
- ✅ Смена выполнена → +1 коин
- ⏰ Опоздал на 15+ минут → -1 коин
- 🎉 Бонус за активность → +0.5 коина
- ❌ Пропустил смену без уведомления → -2 коина

**Пересчёт:** система считает автоматически, но вы можете исправить вручную

### 4️⃣ Система событий (Tribes)

**Где?** Вкладка "События" → "Управление"

**Что это?** Групповые мероприятия для волонтёров (встречи, воркшопы, тимбилдинг)

**Действия:**
- ➕ Создать новое событие
- 📝 Добавить описание, дату, время
- 📋 Указать участников
- ✏️ Редактировать

### 5️⃣ Работа с данными

#### Экспорт в Google Sheets
```
Кнопка "Скачать" → Выбрать Google Sheets
→ Данные обновятся в главной таблице
```

#### Импорт из Google Sheets
```
Если кто-то редактировал напрямую в Sheets:
Нажать "Синхронизировать" → данные обновятся в приложении
```

## 🔍 Часто задаваемые вопросы

**Q: Я изменил смену, но не вижу изменений в Google Sheets**  
A: Нажмите "Синхронизировать" вручную или подождите 5 минут (автосинк)

**Q: Можно ли отменить действие?**  
A: Нет, но вы видите историю всех изменений в каждой смене

**Q: Волонтёр не может выбрать смену**  
A: Проверьте, не заблокирован ли он в списке волонтёров

**Q: Как посчитать коины для волонтёра?**  
A: Откройте его профиль → "Статистика" → там всё автоматически

## 🚨 Важные моменты

⚠️ **Перед удалением** любых данных убедитесь, что это не нужно  
⚠️ **Синхронизируйте регулярно** с Google Sheets (минимум раз в день)  
⚠️ **Дата важна** — убедитесь, что выбрали правильный день  
⚠️ **Время в 24-часовом формате** (14:00, а не 2:00 PM)
