## 📋 Проект

Система управления бассейном для School 21 с поддержкой:
- Волонтёров (обычные, трайб-мастера, проверяющие)
- Учеников и их штрафов
- Google Sheets синхронизации
- Системы вознаграждений (коины)
- Мероприятий (events)

---

## 🎯 Основные требования (от интервью)

### Типы волонтёрства
1. **Обычный волонтёр** - дежурит на смены (4h, 2 смены/день)
2. **Трайб-мастер** - ведёт группу учеников (A, B, C)
3. **Проверяющий** - проверяет групповые проекты на C

### Система штрафов
- **Каждое нарушение = 2 часа отработки**
- **Логика ×2:** Если ученик не пришёл на отработку → штраф умножается
  - День 1: выдан штраф = 2h (multiplier=1)
  - День 2: не пришёл → 2h × 2 = 4h (multiplier=2)
  - День 3: не пришёл ещё раз → 4h × 2 = 8h (multiplier=4)
  - И так далее...

### Волонтёры видят
- **ВСЕ штрафы** (полная прозрачность)
- Список всех учеников со статусом штрафов
- Статистику по часам отработки на ученика

### Team Lead может
- Организовать новый бассейн
- Загрузить смены
- Добавлять/удалять коины волонтёрам
- Видеть рейтинг волонтёров (кол-во смен)
- Синхронизировать с Google Sheets

### Admin
- Всё что Team Lead
- Экспорт данных
- Сброс БД (осторожно!)

## 🛠️ Реализованные фичи (Phase 1)

### Backend (Flask API, порт 5001)

**Модели:**
- `User` - волонтёры/админы
- `Student` - ученики с ник + имя + группа
- `StudentPenalty` - штрафы учеников (2h, 4h, 8h...)
- `Shift` - смены бассейна
- `Event` - мероприятия
- `Reward` - награды волонтёрам

**API Endpoints:**

```
GET  /api/health                    - проверка здоровья
POST /api/auth/login                - вход волонтера
GET  /api/stats                     - статистика
GET  /api/students                  - список учеников со штрафами
POST /api/students                  - добавить ученика
DELETE /api/students/<id>           - удалить ученика
GET  /api/penalties                 - ВСЕ штрафы (видят все)
POST /api/penalties                 - добавить штраф ученику
PATCH /api/penalties/<id>           - изменить статус (done/pending/overdue)
DELETE /api/penalties/<id>          - удалить штраф
GET  /api/events                    - мероприятия
POST /api/events                    - создать событие
PATCH /api/events/<id>              - одобрить событие
POST /api/sync                      - синхронизировать Google Sheets (TODO)
```

### Frontend (React, порт 3000)

**Страницы:**
- 🏠 Dashboard - статистика
- 📅 Shifts - управление сменами
- 👥 Students - **НОВОЕ** управление учениками
- ⚠️ Penalties - **НОВОЕ** система штрафов
- 🎯 Volunteers - список волонтёров
- 🎮 Rewards - награды
- ⚙️ Admin - админ панель

**Стиль:**
- School 21 тема (чёрный + зелёный #00ff00)
- Dark mode by default
- Адаптивный дизайн
- Минималистичный интерфейс

✅ Все кнопки с подтверждением диалогом  
✅ Нет алертов после - только confirm перед
